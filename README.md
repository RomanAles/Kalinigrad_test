# Поиск смысловых копий изображений — Калининград Hackathon

Проект создан для международного хакатона в Калининграде. Цель — находить семантические (смысловые) копии изображений: не просто точные дубликаты, а изображения с тем же содержанием/смыслом (например, разные кадры одного объекта, редизайны, вариации сцены и т.д.).

**Ключевые возможности**
- Поиск изображений по семантической близости (embedding + nearest neighbours).
- Поддержка обработки большего объёма изображений (батчи, индексирование).
- Набор ноутбуков и скриптов для обучения/вычисления эмбеддингов и поиска.

Данный репозеторий - результат моих тестирований и экспериментов
